# 🧮 Modern Glassmorphism Calculator

A sleek, responsive, and functional web calculator built using **HTML5, CSS3, and Vanilla JavaScript**. This project features a modern dark-mode UI with "Glassmorphism" aesthetics and smooth button interactions.

## ✨ Features
* **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
* **Error Handling:** Detects invalid equations and displays an "Error" message.
* **Responsive Design:** Fully functional on mobile, tablet, and desktop screens.
* **Modern UI:** Built with CSS Grid and features a clean, dark-themed color palette.
* **One-tap Delete:** Includes a `DEL` button to fix mistakes character by character.

## 🛠️ Built With
* **HTML5:** Semantic structure.
* **CSS3:** Flexbox and CSS Grid for layout.
* **JavaScript (ES6):** Logic for calculations and DOM manipulation.
